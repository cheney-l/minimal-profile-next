# 个人主页配置说明书（小白版）

这份说明书专门给不懂代码的你。  
你只需要改 **1 个文件**，就能修改网页里的文字、链接、头像、二维码。

---

## 1. 你只需要改哪里？

请打开这个文件：

`D:\代码工作区\minimal-profile-next\app\site-content.js`

以后要改内容，都在这个文件里改。  
**不要改** `app/page.js` 和 `app/layout.js`（这些是页面结构代码）。

---

## 2. 修改前，先做一件事（防出错）

先复制一份备份：

1. 在 `app` 文件夹里找到 `site-content.js`
2. 复制一份，重命名为：`site-content.bak.js`

如果改坏了，可以把备份改回 `site-content.js`。

---

## 3. 每个板块对应哪里改？

`site-content.js` 里主要分成三块：

1. `meta`：网页标题/描述（浏览器标签页会看到）
2. `leftColumn`：左侧个人信息栏
3. `rightColumn`：右侧四个内容板块

---

## 4. 左侧栏怎么改（头像、名字、城市、链接、按钮、二维码）

### 4.1 头像

位置：

`leftColumn.avatar`

可改字段：

- `src`：头像图片地址（最重要）
- `alt`：图片说明文字
- `placeholder`：没有头像时显示的占位文字

示例：

```js
avatar: {
  src: "/avatar.jpg",
  alt: "明灿头像",
  placeholder: "头像占位"
}
```

---

### 4.2 姓名与城市

位置：

- `leftColumn.name`
- `leftColumn.city`

示例：

```js
name: "小小的老子猎猎猎",
city: "Beijing, China",
```

---

### 4.3 左侧链接（GitHub / X / 邮箱）

位置：

`leftColumn.links`（这是一个列表）

示例：

```js
links: [
  { label: "GitHub", href: "https://github.com/你的ID" },
  { label: "X", href: "https://x.com/你的ID" },
  { label: "邮箱", href: "mailto:you@example.com" }
]
```

说明：

- `label` 是显示文字
- `href` 是点击后跳转地址

---

### 4.4 左侧按钮（匿名留言按钮）

位置：

`leftColumn.cta`

示例：

```js
cta: {
  label: "和我聊聊你的团队吧",
  href: "https://你的飞书表单链接"
}
```

---

### 4.5 左侧二维码（微信 + 小程序）

位置：

`leftColumn.qrItems`

示例：

```js
qrItems: [
  { label: "微信二维码", src: "/wechat-qr.png", alt: "微信二维码" },
  { label: "小程序二维码", src: "/miniapp-qr.png", alt: "小程序二维码" }
]
```

说明：

- 如果 `src` 为空字符串 `""`，页面就显示占位文字
- 如果 `src` 有值，就显示真实图片

---

## 5. 右侧四个板块怎么改

位置都在：`rightColumn` 里。

---

### 5.1 第一块：信条

位置：

`rightColumn.belief`

可改字段：

- `indexLabel`：左上角小标题（例如 `01 · Belief`）
- `title`：标题（例如 `信条`）
- `quote`：信条正文

---

### 5.2 第二块：日常出没坐标

位置：

`rightColumn.coordinates`

可改字段：

- `indexLabel`
- `title`
- `lines`：数组，每一项一行
- `note`：下面的小字说明

示例：

```js
coordinates: {
  indexLabel: "02 · Coordinates",
  title: "日常出没坐标",
  lines: ["五道口·知春路·西二旗", "原点大厦·优盛大厦"],
  note: "以及随机刷新在三里屯·望京"
}
```

---

### 5.3 第三块：简介

位置：

`rightColumn.intro`

改：

- `indexLabel`
- `title`
- `body`

---

### 5.4 第四块：Lab

位置：

`rightColumn.lab`

改：

- `indexLabel`
- `title`
- `body`

---

## 6. 图片应该放在哪里？

推荐把图片放到这里：

`D:\代码工作区\minimal-profile-next\public`

如果 `public` 文件夹还没有，就自己新建一个。

然后配置里写：

- `/avatar.jpg`
- `/wechat-qr.png`
- `/miniapp-qr.png`

注意前面有 `/`。

---

## 7. 改完后怎么看到效果？

### 如果网站已经在运行

直接刷新浏览器（`F5`）。

### 如果网站没运行

在终端执行：

```bash
cd /d "D:\代码工作区\minimal-profile-next"
npm.cmd run dev
```

打开：

`http://localhost:3000`

---

## 8. 常见报错怎么处理（小白急救）

### 情况 A：网页打不开（ERR_CONNECTION_REFUSED）

说明服务没启动。重新运行：

```bash
cd /d "D:\代码工作区\minimal-profile-next"
npm.cmd run dev
```

### 情况 B：改完后页面报错

1. 先恢复备份 `site-content.bak.js`
2. 再启动：

```bash
npm.cmd run dev
```

通常是逗号、引号、括号少了一个。

## 9. 你最常改的字段（速查）

1. 网站标题：`meta.title`
2. 左侧姓名：`leftColumn.name`
3. 左侧城市：`leftColumn.city`
4. 左侧按钮文字：`leftColumn.cta.label`
5. 左侧按钮跳转：`leftColumn.cta.href`
6. 头像图片：`leftColumn.avatar.src`
7. 微信二维码：`leftColumn.qrItems[0].src`
8. 小程序二维码：`leftColumn.qrItems[1].src`
9. 信条正文：`rightColumn.belief.quote`
10. 简介正文：`rightColumn.intro.body`
11. Lab 正文：`rightColumn.lab.body`
